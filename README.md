# Student Performance Prediction

Machine Learning project for predicting students' math scores using demographic and academic data.

## Project Overview

This project uses a Linear Regression model to predict math scores based on:
- gender
- parental education
- lunch type
- test preparation course
- reading score
- writing score

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Workflow

1. Data loading
2. Data preprocessing
3. Feature encoding
4. Train/test split
5. Model training
6. Prediction
7. Model evaluation
8. Visualization

## Results

The model predicts students' math scores with reasonable accuracy using Linear Regression.
